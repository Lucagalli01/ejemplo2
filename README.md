# Revista La Fractura

Sitio web de **Revista La Fractura**, medio de comunicación independiente de la Patagonia. Periodismo sin ataduras desde Neuquén hacia el mundo.

## Estructura

```
revista-lafractura/
├── index.html          # Página principal
├── articulos.html      # Listado de artículos
├── columnistas.html    # Columnistas y autores
├── entrevistas.html    # Entrevistas
├── sobre-nosotros.html # Sobre la revista
└── README.md
```

## Deploy

El sitio es estático (HTML/CSS/JS puro), sin dependencias de build.  
Compatible con **GitHub Pages**, Netlify, Vercel o cualquier hosting estático.

### GitHub Pages
1. Subir el repositorio a GitHub
2. Ir a **Settings → Pages**
3. Seleccionar rama `main` y carpeta `/ (root)`
4. El sitio estará disponible en `https://<usuario>.github.io/<repositorio>/`
